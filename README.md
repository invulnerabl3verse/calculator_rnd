# Kalkulator GUI

Prosty kalkulator z GUI (Tkinter), gdzie możesz wpisać własny wzór używając zmiennych `a` i `b`.

Uruchamianie:

```powershell
python c:\Users\Administrator\Documents\vscode_files\calculator_rnd\calculator.py
```

Wzór wpisuj w polu "Wzór"; dostępne są funkcje z modułu `math` (np. `sin`, `cos`, `sqrt`).
